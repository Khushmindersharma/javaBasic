# Java-demo 


## 1. Run in the Local
### Requirement
Installing Java 17:

    sudo apt update
   
1. *** Docker Install ***
    - sudo apt install docker.io

2. *** Docker build from Docker file ***
    - docker build -t java .

3. *** Docker image to container ***
    - docker run -d -p 8080:8080 java 

4. *** The java application running ***
    - take the public ip address and the type after that :8080
    - The application is running on the browser
